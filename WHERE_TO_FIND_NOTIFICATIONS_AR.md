# 📱 أين تجد الإشعارات في التطبيق؟

## ✅ تم التحديث!

الإشعارات الآن موجودة في **ShopDashboard** وستظهر تلقائياً.

## 📍 الموقع في التطبيق

### 1. افتح التطبيق
### 2. سجل دخول كمتجر (مثلاً: M4)
### 3. في الصفحة الرئيسية، اضغط على زر 🔔 **Notifications**

```
┌─────────────────────────────┐
│ 🏠 M4                       │
│ 3 products  ✅ Validé       │
├─────────────────────────────┤
│ 🔔 Notifications  ℹ️ Info   │  ← اضغط هنا!
└─────────────────────────────┘
```

### 4. ستظهر نافذة الإشعارات:

```
┌─────────────────────────────┐
│ Notifications          ×    │
├─────────────────────────────┤
│                             │
│ ┌─────────────────────────┐ │
│ │ Boutique validée ✅     │ │
│ │ Votre boutique "M4"     │ │
│ │ a été validée           │ │
│ │ 25 فبراير 2026    🔵   │ │
│ └─────────────────────────┘ │
│                             │
└─────────────────────────────┘
```

## 🎯 الخطوات بالتفصيل

### الخطوة 1: تسجيل الدخول
```
البريد: m4
كلمة المرور: 123456
```

### الخطوة 2: في الصفحة الرئيسية
- ستجد في الأعلى زر **🔔 Notifications**
- اضغط عليه

### الخطوة 3: عرض الإشعارات
- ستظهر نافذة منبثقة
- تعرض جميع إشعاراتك
- الإشعارات غير المقروءة بخلفية زرقاء

### الخطوة 4: قراءة الإشعار
- اضغط على أي إشعار
- سيتحول إلى "مقروء"
- تختفي الخلفية الزرقاء
